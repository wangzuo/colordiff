# colordiff
wrapper for diff with color output

[![NPM](https://nodei.co/npm/colordiff.png?compact=true)](https://nodei.co/npm/colordiff/)

## Installation
``` sh
npm install colordiff -g
```

## Usage
``` sh
colordiff new original # wrap
diff new original | colordiff # piping
```
